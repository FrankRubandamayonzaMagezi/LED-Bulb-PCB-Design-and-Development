LED Bulb PCB Design and Development

Project Overview



This repository documents the design and development of an AC-powered LED bulb PCB, from schematic capture through PCB layout, fabrication, assembly and testing.



The project is being developed as part of a PCB fabrication and electronics manufacturing internship, following industry-standard design and documentation practices.



Objectives

Design a production-ready LED bulb PCB

Capture schematics using KiCad

Design PCB layouts following DFM principles

Develop custom footprints where required

Document the complete design process

Prepare manufacturing files

Assemble and test the prototype
